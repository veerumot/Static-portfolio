Skills

Cloud

A public cloud, such as AWS & OpenStack, is a type of cloud computing service that provides resources and services to multiple users over the internet. In a public cloud, the underlying infrastructure is owned, operated, and maintained by a third-party cloud service provider.

Containerization

Containerization is a lightweight, portable, and efficient method of packaging, distributing, and executing software applications. Tools used like Docker, Kubernetes, Docker Swarm, Docker Compose.

Deployment

Deployment automation refers to the process of automatically deploying software applications or updates to various environments without manual intervention. Ansible, Concourse, Jenkins, Helm Deployments.

Monitoring and Observability

Monitoring and observability are crucial aspects of managing and understanding the health and performance of software systems. They involve collecting, analyzing, and visualizing data to gain insights into various aspects of an application or infrastructure.

Scripting

Automating in DevOps involves using scripting languages like Python and Shell to streamline and automate various tasks related to software development, deployment, and infrastructure management. Automation is a key practice in DevOps that aims to improve efficiency.

IaC

"IaC" stands for Infrastructure as Code. It is a key concept in modern software development and operations, emphasizing the use of code to automate the provisioning and management of infrastructure resources. using tools like Terraform, CloudFormation Templates.
